# TaskFlow API

TaskFlow API is a task management backend built with FastAPI.

This project is being developed with a focus on clean architecture, API design,
validation, authentication, database integration, testing and deployment.

## Current Status

In development.

## Projects Goals

- Build a backend API using FastAPI
- Apply clean project structure and separation of concerns
- Design RESTful endpoint for users, projects and tasks
- Implement request validation and response model
- Integrate PostgreSQL with SQLAlchemy
- Add authentication and role-based access control
- Prepare the project for deplyment

## Planned Features

- User management
- Project management
- Task management
- JWT authentication
- Role-based access
- PostgreSQL integration
- Modular services
- Automated tests
- Docker support
